Cloud Sync Settings

Connect to a GitHub Gist to share tasks across your team. All team members opening this page will see and edit the same data.

Generate at github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → with "gist" scope only

FIRST-TIME SETUP (5 MINUTES)

  1. Go to github.com → Sign in (or create a free account)
  2. Click your avatar → Settings → Developer settings → Personal access tokens → Tokens (classic)
  3. Click "Generate new token (classic)", name it "STP Planner", tick only "gist" scope, click Generate
  4. Copy the token (starts with ghp_) and paste it above
  5. Leave Gist ID blank and click "Create New Gist & Push Tasks"
  6. Share the Gist ID with your team — they enter it + their own token to connect
